# Project Changelog - Battle Buddy

This changelog documents the public development journey of the **Battle Buddy** project.

---

## [v0.2.6] - 2026-03-29
- **Android 15 Optimized** — Fully optimized for Android 15 edge-to-edge support with transparent system bars for a distraction-free experience.
- **Enhanced Overlay Sync** — Faster and more robust real-time synchronization for guide metadata in the smart overlay.
- **Hardware-Accelerated Caching** — Improved hero and item icon decoding using optimized background scanning to reduce memory footprint.

---

## [v0.2.5] - 2026-03-27
- **Performance** — Refined ad-delivery logic for a smoother user experience.
- **New Hero Content** — Tactical guides and counters for Marcel added to the offline database.

---

## [v0.2.3] - 2026-03-20
- **Data Sync 2.0** — Introduced a new onboarding flow to check for, download, and verify guide updates.
- **Visual Refinements** — Standardized navigation bars and theme-aware UI components.

---

## [v0.0.5] - 2026-02-18
- **Standard Release** — Initial launch of the showcase, featuring dynamic themes and the in-game overlay.

---

**Note**: This changelog focuses on user-facing features and strategic enhancements. Internal code refactors are not documented here for security reasons.
