# Contributing to MLBB Guide Offline

First of all, thank you for considering helping improve this project! This repository is dedicated to project documentation, feature showcase, and community feedback.

---

## 🐞 Reporting Bugs

If you find a bug in the application (e.g., incorrect hero stats, app crashes, or UI alignment issues):

1.  **Search existing issues** within this repository to see if it's already been reported.
2.  If not, [Open a New Bug Report](.github/ISSUE_TEMPLATE/bug_report.md).
3.  Include as much detail as possible: 
    - App Version (vX.X.X).
    - Device Model & Android Version.
    - Steps to Reproduce.
    - Screenshot (if applicable).

## ✨ Suggesting Features

We are always looking to improve the guide and the analyzer!

1.  [Open a Feature Request](.github/ISSUE_TEMPLATE/feature_request.md).
2.  Explain why this feature would be valuable for players.
3.  Describe how you envision it working.

## 📂 Project Structure

To maintain the security and proprietary integrity of the project, the core source code (.java, .xml layout files, build scripts) is **not publicly accessible**. 

However, we value your feedback on:
- **Guide Accuracy**: Is a counter-pick or synergy outdated?
- **UI/UX Feel**: How can we make the overlay more intuitive?
- **Performance**: Is the analyzer fast enough?
